# 🔧 hermes-habit-casino

A [Hermes Agent](https://hermes-agent.nousresearch.com/) skill for **habit-casino** (health category).

RPG gamification engine for your actual life. Dynamic quests, XP, achievements, rivalries, and narrative flavour text. Your agent IS the game master.

## Installation

```bash
cd ~/.hermes/skills/health
git clone https://github.com/Marrowleaf/hermes-habit-casino.git habit-casino
```

## Usage

Once installed, this skill is automatically detected by Hermes Agent.

```
skill_view(name="habit-casino")
```

## Structure

- `SKILL.md` — Full documentation with commands, examples, and pitfalls
- `references/` — Reference materials
- `scripts/` — Utility scripts
- `templates/` — Template files
